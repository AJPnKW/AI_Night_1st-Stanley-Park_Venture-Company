Facilitator support for talk track, demos, timing, and fallback plans.

This page is a concise operational layer for the session facilitator and is intentionally not a main public destination.

Important: this page is independent volunteer material and does not imply official ownership by 1st Stanley Park or Scouts Canada.

Minute-by-minute outline

  • 0-10Welcome, frame safety, define AI plainly.
  • 10-25Explain tool vs model, show prompt quality shift.
  • 25-50Guided prompt practice and review.
  • 50-75Project build time with active coaching.
  • 75-90Share, reflect, and close with verification habits.

Demo sequence

Show a vague prompt, improve it, then compare outputs and ask what changed.

Contingency

If the main AI tool is unavailable, switch to offline prompt drafting and verbal critique.

Tech check

Wi-Fi, projector if used, battery power, browser access, and a printed fallback copy.

Support cues

  • Offer predictable structure and explicit transitions.
  • Keep prompts visible on screen while explaining them.
  • Invite quiet participation without forcing public sharing.
  • Use one instruction at a time when a participant looks overloaded.

Backup prompts

  • Explain a topic to a 16-year-old using 5 bullets.
  • Turn these notes into a one-page checklist.
  • Give me 3 project ideas and rank them by easiest to finish tonight.
  • Rewrite this in simpler language.
