Main message

Three things everyone should know before the session

1

Participants sign up first

Use the participant sign-up page to share device readiness, ChatGPT access status, and one simple build idea.

2

Parents review and approve

Use the parent guide, waiver, consent, and privacy pages to understand the workshop and complete approval.

3

Arrive ready for a supervised workshop

Bring a laptop or Chromebook, keep personal information out of prompts, and follow the shared session plan together.

Parent confidence

Minimal data only

Default sign-up and approval data stays limited to first name, readiness status, age range, simple workshop goals, and parent acknowledgement, with optional email only if needed for follow-up.

Volunteer maintainability

Static site, simple files

No framework is required. The site uses HTML, CSS, and a small amount of JavaScript for navigation state and form configuration.
