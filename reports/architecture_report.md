# Architecture Report

Final build architecture:

- Canonical editable site under `AI_Workshop_4_ventures/web/`
- Static GitHub Pages publish mirror under `github/`
- Shared styles and behavior under `AI_Workshop_4_ventures/web/assets/`
- Shallow public navigation
- Low-profile direct-link volunteer pages
- Documentation under `docs/`
- Repeatable QA under `scripts/`

The architecture favors maintainability, print support, and transparent ownership separation over dynamic features.
