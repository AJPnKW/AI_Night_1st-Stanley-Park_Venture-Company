# Codex Beginner Segment

## Purpose

Give the workshop a safe, short, optional explanation of what Codex or coding-agent style tools are and how beginners should think about them.

## Beginner Framing

- Codex is a helper, not an autopilot
- it can draft code or structured changes quickly
- it can still be wrong, unsafe, or mismatched to the real task
- humans still inspect, test, and decide

## Audience Fit

- best as an optional extension for ages 15 to 18
- especially useful when one or more learners already have some coding exposure

## Delivery Model

- Time required: 5 to 8 minutes
- Delivery mode: optional side segment or extension card
- Facilitator preparation: pick one tiny example, such as a simple HTML tweak or checklist script

## Learner Steps

1. describe the small task clearly
2. ask the tool for help
3. read the result
4. test it
5. fix or reject it if needed

## Risks And Misuse

- overtrust in untested code
- sharing secrets or personal data in prompts
- copying code without understanding what it does
