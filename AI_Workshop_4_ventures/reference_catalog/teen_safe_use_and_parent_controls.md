# Teen Safe Use and Parent Controls

## Purpose

Provide a workshop-local reference for supervised youth use, parent awareness, approval expectations, and minimal-data handling.

## Why This Matters

- the workshop uses ChatGPT with youth
- parent trust depends on clear practical language
- the session must support safe use without becoming alarmist

## Core Model

- supervised use during the event
- no personal or sensitive information entered into AI tools
- minimal readiness data only
- parent review path visible before the event
- age-aware permission handling if required by the tool or provider rules

## What Participants Should Hear

- AI can help, but it can be wrong
- do not enter private details into prompts
- if unsure, ask first
- the workshop is supervised and structured

## What Parents Should Hear

- the activity is supervised
- the focus is responsible use, not unrestricted experimentation
- setup help may be needed at home before the event
- only minimal readiness information is collected

## What Leaders Should Watch For

- private information being typed into prompts
- frustration or freezing
- learners drifting off task
- learners needing a predictable next step

## What The Facilitator Should Control

- safe shared examples
- timing and transitions
- escalation path when a learner is unsure
- explanation of what data is and is not collected
