# Information Architecture

Primary public routing:

- Home
- Participant area
- Parent / guardian area
- Shared FAQ
- Shared resources
- Privacy

Supporting participant pages:

- before
- day-of
- project
- after

Low-profile direct-link pages:

- leader
- facilitator

Legal layer:

- privacy
- disclaimer
- consent

The structure keeps the main public path shallow and predictable while separating volunteer-only operational content from participant-facing routing.
