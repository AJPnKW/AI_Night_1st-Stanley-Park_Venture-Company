# UX Design Principles

- Keep navigation shallow and repeated.
- Put “what next” guidance near the top of each page.
- Use large cards, spacing, and consistent headings to reduce clutter.
- Prefer calm visual contrast over noisy motion.
- Use plain language first, technical depth second.
- Keep role separation obvious.
- Maintain strong focus styles and keyboard support.
- Ensure print output remains useful and clean.
- Preserve a clear independence disclaimer on relevant pages.

The visual system uses yellow, green, and red scarf-inspired accents without implying official branding ownership.
